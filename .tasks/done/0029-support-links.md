# Problem to solve

The web is a set of links between websites. It is important to support the external links from the website.

The Markdown format expected is as below:

```
<http://www.example.com>

[Open in same tab](http://www.example.com)

[Open in new tab](tab:http://www.example.com)

[Internal link](posts/001-bonjour-monde.fr.md)
```

When it is a internal link, it should be converted to the right URL.

The `docs/markdown-cheatsheet.md` must be updated.
