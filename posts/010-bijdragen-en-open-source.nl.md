Pureblog is een Open Source-project onder de MIT-licentie.

Je kunt vrij bijdragen op de GitHub van het project <https://github.com/ostaquet/pureblog>.

Zorg er wel voor dat je de [richtlijnen voor bijdragen](https://github.com/ostaquet/pureblog?tab=contributing-ov-file) respecteert.
